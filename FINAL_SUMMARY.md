## 🚀 How to Get Started

### 3 Simple Steps

**Step 1:** Ensure .env.local has MongoDB URI
```bash
MONGODB_URI=your_connection_string
JWT_SECRET=your_secret
```

**Step 2:** Run the seed command
```bash
npm run seed
```

**Step 3:** Start and explore
```bash
npm run dev
# Visit http://localhost:3000
```
